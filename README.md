# Retail Sales Analytics - SQL Mini Project

A data-engineering focused SQL project built using Northwind retail data.
This project simulates a lightweight ETL pipeline:
raw → staging → star schema → analytics

## Tech Stack
- MSSQL
- SQL
- GitHub

## Project Layers
- Raw data ingestion
- Staging layer (cleaning + transformations)
- Star schema (dim_product, dim_customer, dim_date, fact_sales)
- Analytical SQL queries
- Performance tuning (EXPLAIN, indexing decisions)

## Project Structure
See folders for ingestion, staging, modelling, analytics, and documentation.
